# Morse to LED

Morse to LED is an embedded systems project that converts Morse code input into LED-based output patterns using a microcontroller platform.

It is designed as a practical project for learning electronics, signal encoding, and basic hardware-software interaction.

## Features
- Morse code interpretation
- LED output signaling
- Microcontroller-based implementation
- Educational and easy to extend

## Hardware
Typical hardware used in this type of project:
- Microcontroller board
- LED
- Resistors
- Breadboard and jumper wires

## File overview
- `morse_to_led.ino`: main Arduino-compatible sketch for the project

## How to use
1. Open the `.ino` file in the Arduino IDE
2. Select the correct board and port
3. Upload the sketch to the device
4. Test Morse output through the connected LED

## Learning goals
- Understand Morse code encoding
- Practice digital output control
- Work with basic embedded programming concepts

## Possible improvements
- Add buzzer output
- Add LCD or OLED display
- Add serial input support
- Support decoding as well as encoding
